# ansible_seed
This project is a ansible example

## requirement
- ubuntu 16.04
- ansible 2.2.1.0

## install
- apt-get update
- apt-get install python-pip
- apt-get install libssl-dev
- pip install ansible==2.2.1.0
- apt-get install ansible=2.1.1.0-1~ubuntu16.04.1
